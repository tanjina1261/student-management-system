# Student Management System

A desktop application to manage student records built with Python and CustomTkinter.

## Features
- Add, edit, and delete student records
- Search by name, student ID, or department
- Dashboard with live stats and department bar chart
- SQLite database for persistent storage
- Modern dark UI

## Tech Stack
- Python 3
- CustomTkinter (modern UI)
- SQLite (database)
- Tkinter ttk (table/treeview)

## How to Run
pip install customtkinter pillow
python app.py
